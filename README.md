#Retail Sales Dashboard | Power BI

Project Overview

This project is an interactive Retail Sales Dashboard developed using Power BI. The dashboard provides insights into sales performance, customer segments, regional sales distribution, and top-performing products. It helps businesses monitor key performance indicators (KPIs) and make data-driven decisions.

Objectives

- Analyze overall sales performance.
- Identify top-selling products.
- Compare sales across different regions.
- Study customer segments and their contribution to revenue.
- Track sales trends over time.

Dataset Information

The dataset contains retail sales transaction records including:

- Order ID
- Order Date
- Customer Information
- Product Information
- Category & Sub-Category
- Region
- Sales

Key Performance Indicators (KPIs)

- Total Sales
- Total Orders
- Regional Sales Distribution
- Customer Segment Analysis
- Monthly Sales Trend
- Top 5 Products by Sales

Dashboard Features

1. Sales Overview

Displays total sales and total orders using KPI cards.

2. Regional Analysis

Analyzes sales performance across different regions using interactive filters.

3. Sales Trend Analysis

Visualizes sales trends over time to identify growth and decline patterns.

4. Customer Segment Analysis

Shows the contribution of Consumer, Corporate, and Home Office customers.

5. Product Performance

Highlights the top-selling products based on total sales.
